## 0. One-paragraph state of the program

SNMR is a skeleton-agnostic neural retargeter that maps human motion into 5 humanoid robots (0.34–0.78 m root height, 22–29 DoF) through a single shared per-frame latent z ∈ R^128, distilled from a GMR per-robot IK teacher over 2.48 M paired frames (77 LAFAN1 clips × 5 robots). The core retargeting result is solid and paper-ready: a G1 specialist reaches 2.18 cm MPJPE @100k steps and the shared-5 model 3.2–6.7 cm per robot with zero joint-limit violations, and the shared latent is measurably "aligned, not invariant" across embodiments (linear CKA 0.910; human→robot retrieval R@1 0.749; embodiment identity nonlinearly decodable at 0.909 vs linear 0.278). Everything downstream converges on one bottleneck: foot-skate. The optimizer/solver side is *solved* — a teacher-height oracle contact mask drives skate to 0.0056 m/s with all guards passing — but every *deployable* contact signal fails (under-firing, under-support, or over-density), which closed the entire inference-time correction line, including a full flow-matching side project (E43–E47, closed at its gates) whose deepest finding is that the ambiguity is a Dirac property of the one-teacher data-generating process, not of the model. The two live positives are downstream WBT tracking (B1 reference gate passed at 86% completion, confirmatory matrix promoted) and E48 SSL (co-trained contact BCE structures z to F1 0.257, 11× control and 2× the deployable baseline — the first z readout to beat the mask signal, though only a representation claim so far).

## 1. What We Built

### Core retargeter + analysis infrastructure
- **Model** (`snmr/model.py`): GAT node-shared encoder over the human graph → max-pool → per-frame z ∈ R^128 → optional temporal Transformer (no positional encoding) → AdaLN decoder (graph attention over robot graph, latent re-injected per layer) → qpos + optional contact head. Main model d=128; G1 specialist ≈ 1.6 M params. Encoder input per node = position(3)+6D rotation(6)+velocity(3) in root heading frame + human static graph features; canonicalization removes global XY translation and yaw. Quaternion convention wxyz.
- **Embodiment conditioning**: robot graph read from MJCF (8 static features) → MLP+max-pool embodiment code applied via AdaLN. Joint limits satisfied *by construction* (tanh mapped into MJCF range). Root predicted relative to per-robot-scaled human root in the human heading frame.
- **Training** (`scripts/train_phase1.py`, `train_phase2.py`, `snmr/losses.py`): Phase 1 = G1 specialist; Phase 2 = shared 5-robot model (64-frame windows, K=2 robots/step, L_distill + L_task diff-FK keypoints + L_smooth + L_limits [+ optional L_contact] + symmetric latent-consistency L_z). `zr_decode_prob` wired but deferred.
- **Analysis/metrics** (`scripts/benchmark.py`, `snmr/metrics.py`, `snmr/skeleton.py`/`robot_model.py` diff-FK): E1–E5 representation protocol — embodiment linear probe, motion probe (positive control), MLP attacker + k-class proxy-A-distance (Elazar-Goldberg safeguard), cross-embodiment retrieval, linear CKA, plus a scale-normalized leak-attribution probe. Figures in `runs/figures/` (dual-colored t-SNE/UMAP, CKA heatmap).

### Contact-correction machinery
- **Training-time contact loss** (`decoder.contact_head`, built whenever `contact_weight>0`): bundled BCE + EDGE self-consistency + teacher-mask velocity (E10a); FK-foot-velocity distill term (E25).
- **Inference foot-lock** (`snmr/footlock.py`): source-mask-driven damped-least-squares solver (Jacobian via autograd, line search, limit clamping), interval pinning with 2/3/5-frame blend ramps.
- **Full temporal C6 windowed projection** (`snmr/windowed_contact_projection`): one L-BFGS solve over the full 192-frame window, both leg chains + bounded root XYZ/yaw, exact joint limits, deviation/velocity/acceleration penalties, penalizing the exact mask-gated XY displacement the evaluator scores.
- **Gate-1 factorized soft arms** C0–C4 (control / BCE / BCE+EDGE / stance-velocity / phase-balanced velocity) and **Gate-1b mask/solver arms** M1b (clip-local decoded-ground), M2 (C1-head), M3 (frozen contact-head retrain, only 4 head tensors trainable).
- **Evaluator machinery**: empty-mask windows return undefined (excluded, regression-tested); teacher-height stance-speed aggregation corrected.
- **Contact-mask catalogue**: source/human-flag, teacher-height oracle, decoded/predicted, source-height, decoded clip-local-ground, E10a predicted-head, M3 trained-head.

### Flow / generative side-project machinery (E43–E47, closed)
- **Conditional rectified-flow net** over frozen 128-d latents (`snmr/flow.py`): temporal Transformer (hidden 256, 4 layers, 4 heads, ~1.6 M params), CondOT CFM loss, Euler sampler (NFE≈25) with SafeFlow-style endpoint guidance and differentiable physics cost (skate/penetration/smoothness) through the frozen decoder + FK.
- **Trainer/evaluator** (`scripts/train_latent_flow.py`, `eval_latent_flow.py`): 20k steps, batch 8×64 frames, ≈15 min on A10G; guidance post-mortem probe (`diagnose_latent_flow_guidance.py`); v2/v3 substrate tooling (`train_v0_noise_decoder.py` noise augmentation, `probe_multisolution_diversity.py`); `tests/test_flow_retarget.py` (10 tests, green).

### WBT downstream + latent-command infra
- **From-scratch WBT training** on the pinned Holosoma clone (monkeypatch-free `func=` override), g1-29dof-wbt robot, MuJoCo/Warp, 1,024 envs, PPO, 8k iters. SNMR/GMR qpos @30 Hz → shared resampler → 50 Hz → MuJoCo FK → WBT NPZ → Holosoma PPO (source-blind after export).
- **Latent bridge** (`scripts/export_wbt_with_latent.py`, `snmr/integration/wbt_latent.py`): time-aligned latent_z; observation interfaces `snmr_latent`, `latent_preview_command` (L1 actor, 480-d), `motion_command_with_latent_preview` (critic, 670-d), `motion_command_with_explicit_preview` (C1/C3, 212-d).
- **Phase-3 multi-clip infra** (`export_wbt_gmr_latent_batch.py`, `MultiMotionLoader`): train set 75,184 frames / 6 walk-family clips / ~25.1 min + held-out (walk1_subject5, run2_subject1). Idempotent resumable drivers with provenance sha256, official terminal-aware evaluator (100 phase-stratified 10-s windows), paired-bootstrap CIs.
- **B1 reference-dev gate** artifacts (`runs/wbt_reference_dev/`) and `scripts/probe_latent_contact.py` (E38).

### E48 SSL representation tooling
- **Input-corruption pretext** (`corrupt_input_features()`, MotionBERT pattern): additive Gaussian, per-(frame,joint) zeroing, whole-frame zeroing; encoder inpaints against clean distillation target.
- **Co-trained contact BCE head** (`contact_bce_weight=0.5`, teacher-height hysteresis enter 0.03/exit 0.05), trained jointly (not frozen retrofit); `objective_manifest()` records both objectives.
- **Probe harness** (`probe_latent_contact.py`): held-out-clip logistic regression on z (`z_linear`) and z±1-frame context, controls source-height mask (deployable baseline) + majority.
- **2×2 A/B/C/D design**: A=distill-only, B=denoise, C=denoise+BCE, D=BCE-only; all large config, lr 3e-4, 30k steps, seed 0, window 64, d=128.

## 2. What We Learned

### Retargeting fidelity & sharing
- **Fidelity (C1, DONE)**: G1 specialist 2.18 cm MPJPE @100k / 3.75 cm @50k (sparse held-out); 3.1 cm on dense held-out (abstract). Shared-5: G1 6.67, T1 5.80, N1 5.54, PM01 5.72, Toddy 3.16 cm. Zero limit violations; SNMR better than teacher on limit-proximity margin (0.29 vs 0.57).
- **Sharing cost (C3, DONE)**: ~1.4 cm at matched schedule (specialist 3.75 cm → shared 5.12 cm on matched-G1). No pervasive gradient mean-conflict; sporadic negative observations remain.
- **Zero-shot embodiment transfer (C4, DONE, NEGATIVE)**: unseen LORO PM01 = 29.6 cm vs 5.7 cm in-training = 5.2× error; fails at current scale.
- **Robot→robot transfer (C7, PEND, FAILS)**: 24–45 cm because the decoder was never trained on robot encodings (E19); decode-from-z_r augmentation (E21) proposed.
- **Temporal Transformer not load-bearing**: removing it *improves* MPJPE 4.71→3.95 cm and skate 0.386→0.358 m/s while worsening jerk 614→720 — order-aware positional encoding needed before any temporal claim.

### Latent representation
- **Aligned, not invariant (C2, DONE)**: linear CKA 0.910 (0.86–0.98 over 15 pairs; adult humanoids 0.96–0.98, human↔Toddy weakest 0.82); retrieval R@1 0.749 / MRR 0.838 (chance 0.010); embodiment identity linear probe 0.278 vs MLP attacker 0.909, proxy-A-distance 1.78.
- **Content-specific, not semantic**: motion-category probe ~chance (0.151 / 0.152) despite 75% exact-clip retrieval → instance-level trajectory detail, no linearly-separable category structure. Height-normalization leak intervention moves the attacker only 0.943→0.933 (OOD; rules out simple amplitude normalization but does not causally attribute the leak).

### Contact / foot-skate
- **Root cause (C5, E24/E26)**: skate is smooth XY *oscillation* (~2.9 cm RMS, τ≈0.1 s) of a correctly-placed stance foot, not drift or high-frequency jitter. Skate tracks MPJPE at a ~7–9 s⁻¹ ratio constant across all checkpoints — the velocity shadow of regression error. Detection *under-fires*: decoded contact frac 0.33 vs teacher 0.74 (58% agreement); under the 0.3 m/s gate 0.03 vs 0.29.
- **Training-time levers insufficient**: bundled contact loss made skate *worse* (w0.2 → 0.497 m/s; w0.1 → 0.402, no gain). Velocity-distill helps but not enough (w10: teacher-height stance 0.417→0.304 m/s, still 3.8× the ≤0.08 endpoint). Gate-1 soft arms improve consistently but fail the endpoint at all 3 seeds: 3-seed means C0 0.701, C3 0.469, C4 0.264 m/s (C4 −62.4% speed, MPJPE −1.61 cm passing 4.0 cm) — none reaches ≤0.08 in any seed.
- **Oracle-vs-deployable is the whole story (decisive C5 result)**: teacher-height oracle passes every guard — DLS 0.502→0.047 m/s (MPJPE 3.70 cm, jerk 727); full C6 0.502→0.0056 m/s (MPJPE 3.82 cm, jerk 697). Every deployable mask fails: source-contact C6 0.099 m/s but MPJPE 4.28 cm (speed + rel-MPJPE guards fail); source-contact DLS 12-iter reaches 0.068 m/s but MPJPE 4.08 cm and jerk +35%; a skate↔jerk Pareto (blend2 0.068/1.35× → blend5 0.132/1.13×) offers no free lunch. **Verdict: mask precision, not the solver, is the bottleneck; mask/solver iteration CLOSED.**
- **Gate-1b closed on density/support**: M1b under-supports (58/16,128 samples, 34/42 zero-support windows, F1 0.0201, projection≈identity); M3 over-fires (prevalence 0.715 vs oracle 0.0887, precision 0.096, F1 0.324, MPJPE 3.66→6.51 cm, fails jerk + penetration). Both deployable extremes fail; the oracle sits between.
- **Contact not deployably decodable from the frozen distillation latent (E38)**: held-out z probe F1 0.044 (0.052 with ±1-frame context) vs 0.127 source-height mask; walk1 z-AUROC 0.51–0.64.

### Generative-retrofit failure chain (flow side project, CLOSED)
- **F0/E43 FAIL**: Dec(z_r) = 50.06 cm vs 3 cm bar — nothing ever trained decoding-from-robot-encodings; transport degenerates to identity.
- **F1/E44 PASS**: unguided flow reproduces the decode (3.49 cm vs raw 3.48 cm) — a generative latent head is free.
- **F2/E44 FAIL (stop rule)**: guidance nearly inert; best oracle cell 0.400→0.339 m/s vs 0.08 endpoint; z-descent control 5× better at 0.221 m/s but still ~2.5× off.
- **E45 structural diagnosis**: (1) SCALE — clamped push ~0.15% of velocity norm (600× below parity), but fixing it is insufficient; (2) CONTRACTION/Dirac — transport retains 9% of a u=0.2 perturbation, endpoint diversity 0.4% of ‖z_h‖, a Dirac conditional p(z1|c)=δ_c has no distribution to steer; (3) ORTHOGONALITY — norm-matched pushes (ρ≤1.0) make skate *worse* and guidance is ~orthogonal (cos≈0.05) to Adam-descent.
- **E47 local-uniqueness (deepest finding)**: locally corrected solutions around a single IK teacher are near-unique (manufacturability 3/21 windows vs 60% bar; diversity 0.0019 rad vs 0.01 rad floor). **The Dirac conditional is a property of the data-generating process, not the model** → multimodality must enter at the teacher level (main-line Track A5). Substrate noise-finetune dead in all three variants (0.224 / 0.219 / 0.221 m/s vs required ≤0.166); decodable z_r improved 50→19 cm but still 6× off.
- **Method critique**: CORRECT = CondOT path, velocity target, CFM loss, Euler integration, endpoint-prediction guidance, preregistered gates+null control; WRONG = fitting a generative model to a deterministic (target=condition) dataset, guiding in a raw regression latent, and a dimensional-analysis-free scale transplant of SafeFlow's α∈[500,10000] with ±0.2 clamps. Future guidance must be relative, never absolute.
- **Literature convergence**: Feng ICML'25 proves gradient guidance g ∝ Σ_{1|t}∇J → with δ_c conditional Σ→0, so guidance is *provably* squeezed to zero (independently rediscovers E45 contraction). MLD's own ablation (unregularized-AE latent FID 0.473→5.033) is the mechanistic twin of the E43/E44 collapse. z-descent's 5× advantage is predicted by the OC-Flow optimal-control family.

### Downstream WBT tracking
- **B1 reference gate PASSES (2026-07-20)**: from-scratch 8k SNMR-reference walk1 policy = 0.86 completion (floor 0.70) → **promote_confirmatory_matrix**. SNMR 0.86 / 8.8968 s / 0.2502 rad vs GMR context 0.88 / 8.9528 s / 0.2543 rad (SNMR −2 pp completion, −0.056 s survival, but −0.00415 rad joint RMSE, i.e. slightly *better* on RMSE).
- **Reference-quality skate is a live caveat**: aligned SNMR-vs-GMR body MPJPE mean 3.34 cm, DOF error 0.0509 rad, but height-mask stance speed 0.432 m/s (SNMR) vs 0.123 m/s (GMR), slide fraction 0.572 vs 0.089.
- **E35 budget**: walk completion 3%/48%/88% at 2k/4k/8k iters (8k valid for single-clip walk only); dance 0%/2%/10%, fight no 8k checkpoint (excluded from Phase 3).
- **E34 was undertrained control**: both GMR and SNMR 0.0% 10-s completion (missed 50% floor); SNMR joint error favorable in 8/9 matched pairs — verdict "undertrained," not equivalence/benefit.

### SSL latent structuring (E48, single seed)
- **Co-trained contact BCE structures z**: z-linear F1 A 0.023 / B 0.099 / C 0.174 / D 0.257. **D (BCE-only) = 10.99× control and 2.03× the deployable source-height baseline (0.127)** — the first z readout to beat the mask signal, and the mirror of the M3 lesson (the same BCE that failed as a frozen retrofit works when co-trained; replicates HuMoR's 97%-accuracy co-training pattern).
- **Corruption helps but is sub-additive**: B (denoise) = 4.23× control; but C 0.174 < D 0.257 — the two pretexts are not additive at this scale.
- **Fidelity guard fails in every aux arm**: registered guard val MPJPE ≤ A+0.3 cm; verdict-table 30k MPJPE A 7.15 / B 9.36 / C 11.15 / D 10.30 cm (+2.2 to +4.0 cm). Representation claim only; F1 0.257 still far below mask-precision needs (cf. M2 head F1 0.87 still failed as a mask via density).

## 4. The Through-Line

**One bottleneck, restated five ways: the deployable contact signal.** The retargeter's pose fidelity is not the problem — MPJPE is 2.18–6.7 cm with zero limit violations, and pose metrics are simply blind to the artifact (MPJPE is fine at 5× teacher skate). The problem is that a correctly-placed stance foot oscillates ~2.9 cm RMS in XY, and the skate/MPJPE ratio (~7–9 s⁻¹) is constant across every checkpoint — foot slide is the *velocity shadow of the same regression error*, not a separable failure mode. Five independent corrector families (windowed projection, DLS foot-lock, latent Adam descent, guided flow, and the training-time soft arms) *all pass* with a teacher-height oracle contact mask and *all fail* with any deployable mask. The mask under-fires (decoded 0.33 vs teacher 0.74), under-supports (34/42 zero-support windows, F1 0.02), or over-fires (M3 prevalence 0.715, F1 0.32); the oracle sits in a precision band no deployable estimator has reached. Mask precision, not the solver, is decisively the binding constraint.

**Why the mask can't be recovered from the frozen model: MSE conditional-mean / Dirac, and "aligned not invariant."** MSE distillation of a single IK teacher is MLE under a unimodal Gaussian and provably interpolates modes — the literature-standard explanation of exactly this foot-skate signature. Downstream, the frozen latent inherits this: contact is not linearly decodable from z (F1 0.044 vs 0.127 baseline), and the flow project proved the deeper reason — the learned p(z1|c) is a Dirac δ_c, so there is no distribution to steer within (guidance is provably squeezed to zero as Σ_{1|t}→0, and empirically retains 9% of perturbations with cos≈0.05 to the productive direction). E47's local-uniqueness result elevates this from a model artifact to a **property of the data-generating process**: locally corrected solutions around one teacher are near-unique. The latent being "aligned not invariant" (CKA 0.91 yet identity nonlinearly decodable at 0.91) was the hopeful premise for a non-Dirac p(z_r|z_h), but it failed the decodability gate first. Retrofitting generation onto a deterministic target is structurally doomed — MLD's own FID 0.473→5.033 collapse is the mechanistic twin.

**Where the surviving methods converge: physics-executed supervision + co-trained auxiliaries.** Every closed line points the same direction. The one durable fix for the Dirac is to inject genuine multimodality at the *teacher* level — multiple IK configs or, more promisingly, simulator/RL-repaired supervision (main-line Track A5, PULSE-adjacent: valid latents come from simulator-executed actions, not kinematic MSE). That lever is gated on a competent WBT policy, which the B1 gate (86% > 70%) has just unblocked by promoting the confirmatory matrix. Independently, E48 shows the *other* convergent ingredient works today: a **co-trained** contact BCE head (the HuMoR pattern) puts contact into z at F1 0.257 — 11× control, 2× the deployable baseline — precisely the intervention that failed as a frozen M3 retrofit. The synthesis every verified thread agrees on for any future generative retargeter: a co-trained generative head + co-trained contact/velocity auxiliaries + masked-denoising encoder pretext + simulator-executed supervision targets. The program has thus mapped the full negative space (retrofit correction on a frozen MSE latent is closed) and localized the two positive levers (physics-executed supervision, co-trained auxiliaries) that have not yet been combined at full budget.

## 5. Open Questions Carried Forward (ranked by leverage)

1. **Simulator-derived / physics-repaired supervision** — the single lever every corrector shares (oracle passes, all deployable masks fail; E47 says multimodality must enter at the teacher). Now unblocked by the B1 pass but still needs the WBT teacher matured beyond walk. *Highest leverage; remains UNVERIFIED in literature after two rounds — main-line A5 rests on the in-house oracle-pass evidence + PULSE adjacency, first-principles.*
2. **Confirmatory WBT source matrix (promoted)**: is SNMR-reference physically non-inferior to GMR under *matched* from-scratch training (seeds 0/1/2 × eval 404/405)? B1's GMR was a continuation run, so non-inferiority is descriptive (−2 pp completion, but −0.00415 rad RMSE) not yet established.
3. **E48 fidelity price vs convergence lag**: is the +2.2 to +4.0 cm guard failure at 30k real or slower convergence (control 7.15 cm @30k on a schedule reaching ~2.2 cm @100k)? Needs a full-budget (100k) D-arm replication with seed variation *(RUNNING as of 2026-07-20, `runs/e48_ssl_100k/`)*.
4. **Deployable mask from a co-trained head**: E48 proves co-training structures z (F1 0.257) but declines the deployable-mask claim. Would a contact head trained with the *corrected* teacher-height mask at C1 weight (0.25) on an E03-class specialist beat the source mask? The registered right test, never run.
5. **C3 sharing cost**: can a wider trunk or rank-8 per-robot decoder adapters close ≥half of both the mean and worst-robot specialist gap within ~20% robot-specific params (before PCGrad)?
6. **C7 robot→robot transfer (24–45 cm)**: does the `zr_decode_prob` decode-from-z_r augmentation (E21) fix it?
7. **Zero-shot embodiment (C4, 5.2× negative)**: do synthetic morphology variants + morphology-complete features (mass/inertia, actuator limits, link scale, contact geometry) + few-shot adapters generalize to multiple held-out robots?
8. **Temporal modeling**: does an order-aware positional-encoding transformer help, given no_temporal currently wins? Required before any temporal claim.
9. **Per-clip PPO budgets for dance/fight-class clips** (dance 10%, fight no 8k checkpoint) — multi-motion training may itself need fixing before Phase-3 latent-arm failures are interpretable.
10. **Load-bearing latent for tracking** (Track B / Phase 4): can a causal, control-trained command code beat explicit commands, versus the current frozen distillation latent that only reaches L1 feasibility (72–77% completion, 8–18 pp below explicit)?
11. **Held-out rigor**: walk1_subject5 is policy-held-out but researcher-development-seen — a clean generalization claim needs a reserved SNMR validation clip (run2_subject1 flagged as cleaner primary).

*Deprioritized / transferable*: the OC-Flow / D-Flow optimal-control corrector formulation (predicted to beat per-step nudging, never empirically tested here); the V4/W3 flow-guided-reference WBT arm (never armed).
